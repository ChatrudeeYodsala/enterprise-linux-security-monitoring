# Lynis Security Assessment

## Objective

Perform a security assessment of the Ubuntu Server using Lynis.

## Environment

- Ubuntu Server 24.04.4 LTS
- Lynis

## Verification

- Installed Lynis
- Completed a full system security audit
- Reviewed Hardening Index
- Analyzed security recommendations

## Result

The system was successfully assessed, providing actionable recommendations for improving overall security posture.

## Lessons Learned

Security assessment tools help identify configuration weaknesses and provide recommendations for continuous hardening.
