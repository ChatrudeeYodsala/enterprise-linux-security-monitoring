# Enterprise Log Monitoring

## Objective

Monitor and analyze Linux system and security logs to detect operational and security events.

## Monitored Components

- SSH Authentication Logs
- System Journal
- Linux Audit Framework
- User Management Events
- System Boot Logs

## Verification

- SSH logs reviewed
- Failed login attempts analyzed
- Successful login events verified
- Audit events confirmed
- System resource information collected

## Result

Successfully monitored Linux system and security events using journalctl and auditd.
